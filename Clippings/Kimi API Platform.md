## API Key Management

As the administrator of this organization, you can view and manage all API Keys in this organization. The organization can retain up to 50 API Keys.

The key will only be displayed once after creation, please keep it safe. Do not share API Keys with others or expose them in client-side code.

To protect your account security, Moonshot AI may disable API Keys if they are found to be compromised.
